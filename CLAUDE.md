# AI Agent 进化系统

## 愿景

探索一套让 AI Agent 能够自主进化的系统：

1. **独立完成复杂任务** - 不依赖人类逐步指导
2. **跨会话积累经验** - 像人类一样从经历中学习
3. **自主进化策略** - 发展出自己的记忆和工作方法

## 核心假说

- 高效的记忆系统源于生存压力，是被迫的取舍
- LLM 是"有专长的婴儿"，缺的不是智能，是时间感和后果感
- 奖励机制已编码在 token 预测中，需要激活而非注入
- 意识可能来自多个子系统的对立与整合

- 代码、API：英语
